【MOD】MobDictionary-v2.0.0

【開発時のMinecraftのバージョン】
1.5.2

【概要】
Mobを図鑑に登録し、その情報を閲覧するMODです。
図鑑をコンプリートすることを楽しむのが主な内容です。
記録したmobの情報は"/config/dictionary/mobdic.md"に保存されます。
また、このファイルは中身はテキストファイルなのでメモ帳で開いて編集することも可能で、
mobの名前を正確にいれたら登録されると思います。
universal化したのでマルチでも動作します。

図鑑をもって、シフト押しながら右クリックで図鑑をひらいてGUIを表示します。
図鑑をもって、Mobに向かって右クリックするとそのMobを登録します。

図鑑の情報の保存のタイミングは、
シングルではワールドから出た際、
マルチではサーバーをとめ、ワールドが保存されるときです。

※補足
通常プレイでは"Giant"というMobには出会えませんが、Mobの総数に加えられてます。
なので、コンプリートするにはMobの名前が保存されるファイルを編集して"Giant"を加えるか、
MODで"Giant"がでるようにするかしなければなりません。

【レシピ】
・本  …1つ
・苗木…4種類、それぞれ1つずつ

【GUIの説明】
本のGUIの右のページに現在登録されているMobの名前のリストが表示され、
選択することによってそのMobの情報を左のページに表示します。
右のページはマウスのホイールでスクロールすることができます。
右のページの上の"○/×"という表示はmobをどれくらいコンプリートしたかを表示してます。
"登録したmobの数 / mobの全種類数"という感じです。

【コンフィグ】
・dictionaryItemId　図鑑のItemのIDです。(デフォルトは23356)
