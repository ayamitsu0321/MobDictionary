【MOD】MobDictionary_1.3.1_v0.0.2

【概要】
mobを記録する図鑑を追加します。
図鑑をコンプリートすることを楽しむのが主な内容です。
記録したmobの情報は"/config/dictionary/mobdic.md"に保存されます。
ワールド間でも情報は共有されるので注意です。
また、このファイルは中身はテキストファイルなのでメモ帳で開いて編集することも可能です。
mobの名前を正確にいれたら登録されると思います。

【レシピ】
・本…1つ
・苗木･･･4種類それぞれ1つづつ

【GUIの説明】
本のGUIの右のページがmobの登録されてるmobのリストです。
右のリストはマウスのホイールでスクロールすることができます。
左のページはmobの情報を表示します。
右のリストからmobを選ぶと左のページに表示されます。
Nameは名前、IDはMinecraftの内部で設定されてるID、Healthはmobの最大体力です。
右のページの上の"○/×"という表示はmobをどれくらいコンプリートしたかを表示してます。
"登録したmobの数 / mobの全種類数"という感じです。

【コンフィグ】
・図鑑のItemID(23356)を変更可
・registerGiantをtrueにしたら図鑑に"Giant"が登録されます。通常では登録できないので。

・isKeySetをfalseにしたら(デフォルト)
	・アイテムの図鑑を追加します
	・その図鑑をもってmobを叩くと登録します(叩いてもダメージ、ノックバックなし)
	・図鑑を持って右クリックでGUIを開きます

・isKeySetをtrueにしたら
	・キーボードの"B"のキーで図鑑のGUIを開くことができる
	・mobを攻撃することによって図鑑に登録する
	・アイテムの図鑑は追加されない
